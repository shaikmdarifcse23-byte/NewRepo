# React Portfolio Starter

A beginner-friendly React portfolio starter built with Vite, React Router, and reusable functional components.

## Folder structure

- `package.json` - Project dependencies and scripts
- `vite.config.js` - Vite configuration for React
- `index.html` - Root HTML file
- `src/`
  - `main.jsx` - App entry point
  - `App.jsx` - Main app component with routing
  - `App.module.css` - Global styles and layout
  - `components/`
    - `Navbar.jsx` - Responsive navigation bar
    - `Footer.jsx` - Footer component
    - `Navbar.module.css` - Navbar styling
    - `Footer.module.css` - Footer styling
  - `pages/`
    - `Home.jsx` - Landing page
    - `About.jsx` - About section
    - `Skills.jsx` - Skills showcase
    - `Projects.jsx` - Projects list
    - `Contact.jsx` - Contact section
    - `Page.module.css` - Shared page styles

## Installation

1. Open a terminal in this project folder.
2. Run:

```bash
npm install
```

## Available commands

- `npm run dev` - Start the development server
- `npm run build` - Build the production bundle
- `npm run preview` - Preview the production build locally

## Notes

- Update the `Contact` page with your real email and GitHub URL.
